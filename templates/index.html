How to use Spaceship API

  • /api/v1/past - API data for launched rockets in the past1
    • parameters: org, rocket, success
    • org: Organization name
    • rocket: name of rocket
    • success: True or False
    • Usage: /api/v1/past?org=spacex&success=true
  • /api/v1/upcoming - API data for upcoming rockets
    • parameters: org, rocket
    • org: Organization name
    • rocket: name of rocket
    • Usage: /api/v1/upcoming?org=spacex
  • /api/v1/rocket - API data for rocket data
    • parameters: mfr, rocket, status
    • mfr: Manufacturer name
    • rocket: name of rocket
    • status: current status of rocket - planned, active or retired
    • Usage: /api/v1/rocket?mfr=spacex&status=planned
